## Contributing to Changelog

When contributing to HabitFlow, please:

1. **Add entries** to the `[Unreleased]` section
2. **Use categories**: Added, Changed, Deprecated, Removed, Fixed, Security
3. **Link to issues**: Reference GitHub issues where applicable
4. **Be descriptive**: Help users understand the impact of changes

Example entry:
```markdown
### Added
- New forest theme with seasonal changes [#123](https://github.com/MSCRAFI/habitflow/issues/123)
- Habit reminder notifications via email [#124](https://github.com/MSCRAFI/habitflow/issues/124)

### Fixed  
- Streak calculation bug for timezone differences [#125](https://github.com/MSCRAFI/habitflow/issues/125)
```
